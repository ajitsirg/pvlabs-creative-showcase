

# PV Labs — Creative Digital Agency Website

## Overview
A premium, dark-themed, long-scroll website for PV Labs creative agency. Built with React, Tailwind CSS, and Framer Motion. Deep midnight palette with Electric Violet and Royal Blue accents, glassmorphism cards, and bold geometric typography.

---

## Pages & Structure

### 1. Home Page (Long Scroll — 14+ Sections)
- **Hero**: Massive 6rem+ headline "We build the digital artifacts of tomorrow", gradient text, animated CTA button, device mockup visual
- **About Preview**: Agency intro with stats counters (500+ projects, 200+ clients, 8+ years)
- **Services Grid**: 4-column bento grid showing all 22 services as glass cards with icons and hover lift
- **Featured Work**: Full-bleed portfolio gallery with category tags, case study previews
- **Design Process**: 5-step timeline (Discovery → Strategy → Design → Develop → Launch)
- **Testimonials**: Client quotes in glass cards with avatars and company names
- **Why Choose Us**: Icon-driven value propositions grid
- **Industries We Serve**: Visual grid (Tech, Healthcare, Finance, E-commerce, Education, etc.)
- **Team Introduction**: Photo cards with roles, social links
- **Client Logo Slider**: Animated marquee of partner logos
- **Awards & Certifications**: Achievement badges section
- **Blog Preview**: 3-card latest articles grid
- **FAQ Accordion**: Common questions with smooth expand/collapse
- **Contact CTA Banner**: Gradient banner with "Start Your Project" CTA
- **Footer**: Navigation links, social icons, newsletter signup

### 2. Services Page
- Hero with service count and intro
- Individual detailed sections for all **22 services**, each containing:
  - Service title + icon
  - Description + key benefits list
  - Visual mockup/placeholder image
  - CTA button
- Services organized by category (Branding, Print, Digital, Motion, Marketing)
- Pricing packages section (Basic, Pro, Enterprise glass cards)

### 3. Portfolio Page
- Category filter tabs (All, Branding, Web, Print, Motion, Marketing)
- Masonry/grid gallery with hover overlays
- Case study preview cards with project details
- Before/After comparison sections
- Project descriptions with client info

### 4. About Page
- Agency story with timeline
- Mission & Vision statements
- Team member grid with bios
- Tools we use (icon grid)
- Work process visualization
- Achievements & milestones
- Client logos section

### 5. Contact Page
- Contact form (name, email, service dropdown, message, budget range)
- Quick quote form
- Map placeholder section
- Email, phone, address info cards
- Social media links
- Office hours

## Design System
- **Background**: Deep Midnight (`240 10% 3.9%`)
- **Primary**: Electric Violet (`263 70% 50%`)
- **Accent**: Royal Blue (`217 91% 60%`)
- **Typography**: Plus Jakarta Sans (headings), Inter (body)
- **Cards**: Glassmorphism with inner-glow shadows, `rounded-[24px]`
- **Buttons**: Gradient violet-to-indigo, rounded-full, glow shadows
- **Motion**: Framer Motion entrance animations, hover lifts, staggered reveals

## Navigation
- Fixed glass navbar with logo, page links, and "Get a Quote" CTA
- Mobile hamburger menu with slide-in drawer
- Smooth scroll for home page sections

## Extra Features
- Newsletter signup in footer
- Instagram feed preview grid
- Design workflow timeline visualization
- Responsive across desktop, tablet, and mobile
